# App Jurídica — Gestión Jurídica Integral

SaaS de gestión jurídica para abogados argentinos. Combina gestión de expedientes, asistencia con IA y consulta automática de portales judiciales.

## Stack

- **Frontend:** React + TypeScript
- **Backend:** Node.js + TypeScript
- **Base de datos:** PostgreSQL
- **IA:** API de Anthropic (Claude)
- **Almacenamiento:** Compatible con S3

## Cómo ejecutar

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend (en otra terminal)
cd frontend
npm install
npm run dev
```
